``steam.ext.commands`` -- Bot commands framework
================================================

This extension offers the ability to easily create Steam bots using a similar system to that of
`discord.py <https://github.com/Rapptz/discord.py>`_ of which this is heavily inspired by. It aims to make creation of bot easy.
